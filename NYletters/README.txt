How to use the program:
1. In src/main/resources replace the file with a new csv containing database info about centers.
2. Run centers/CenterListParser.java. Get the list of all centers in outputs/centers/Centers.txt.
	Copy the content to Word file and format it. 
	NOTE! csv file may contain incorrect letters (broken foregn symbols). Please check the output list
	before printing it.
3.Run addresses.AddressParser.java. Get the file with addresses for 
	envelopes: outputs/addresses/Addresses.txt. Copy the content and paste to Word file. Format it.
	NOTE! csv file may contain incorrect letters or not full info. Please check the output addresses
	before printing it.
